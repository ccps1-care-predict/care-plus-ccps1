# CarePredict — Protótipo Navegável

Abra o arquivo `index.html` em um navegador.

## Estrutura
- `index.html` — telas e navegação
- `styles.css` — estilo visual
- `script.js` — troca de telas e modo paciente/médico

## Cobertura funcional
Protótipo baseado no escopo funcional do MVP do CarePredict:
- Paciente: dashboard, recomendações, agendamento, histórico, wearables
- Médico: dashboard, pacientes, ficha clínica, estilo de vida, consulta
